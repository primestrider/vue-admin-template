# Admin Template Vue using Nuxt UI

Modern admin template built with **Vue 3 + Vite + Nuxt UI** for scalable and modular business dashboards.

---

## Tech Stack

- Vue 3 (Composition API)
- Vite
- Nuxt UI
- TailwindCSS v4
- TypeScript
- Pinia
- Vue Router
- TanStack Vue Query
- Axios
- vue-i18n

### installation

```
npm install
```

### development

```
npm run dev
```

### build

```
npm run build
npm run preview
```
